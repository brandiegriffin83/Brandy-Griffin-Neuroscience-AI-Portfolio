# A03 – Bio-Inspired AI Audit: ChatGPT vs Human Brain

## Problem
The purpose of this assignment was to evaluate ChatGPT by comparing its performance to key neuroscience concepts such as parallel processing, memory, and attention. The goal was to understand how artificial intelligence systems function in relation to how the human brain processes information.

## Approach
To complete this assignment, I conducted a series of experiments using ChatGPT. Each experiment was designed to test a specific cognitive function:

- Parallel Processing Test: I gave ChatGPT multiple tasks in one prompt to see how it handled them.
- Memory Recall Test: I tested whether ChatGPT could remember information from earlier in the conversation.
- Selective Attention Test: I gave multiple instructions but included one final instruction to see if ChatGPT would prioritize it.

These experiments were based on neuroscience concepts learned throughout the course.

## Results
The results showed that ChatGPT was able to successfully complete all tasks given. However, it processed the tasks sequentially rather than simultaneously, which is different from how the human brain performs parallel processing. 

In the memory test, ChatGPT was able to recall previously given information accurately, but this was due to its ability to reference earlier parts of the conversation rather than true memory storage. 

In the attention test, ChatGPT correctly followed the most important instruction, demonstrating its ability to prioritize tasks based on input structure.

## What I Learned
From this assignment, I learned that artificial intelligence can simulate many aspects of human cognition, but it does not function the same way as the human brain. The brain uses biological systems to process information, while AI relies on pattern recognition and data. 

This assignment helped me understand both the strengths and limitations of AI systems and how neuroscience can be used to improve future AI development.

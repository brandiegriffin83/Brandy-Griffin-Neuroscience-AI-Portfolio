# A04 – Spiking Neurons with Wolfram

## Problem
The purpose of this assignment was to explore how biological neurons behave and how their activity can be modeled using mathematical equations. The focus was on understanding neuron firing behavior using the Leaky Integrate-and-Fire (LIF) model.

## Approach
To complete this assignment, I used Wolfram Language to simulate neuron activity. I defined neuron parameters such as membrane potential, threshold, and input current. I then used differential equations to model how the neuron responds over time.

I also created visualizations and an interactive simulation to observe how changes in input current affect neuron firing behavior. This allowed me to better understand how neurons generate spikes.

## Results
The results showed that neurons fire when the membrane potential reaches a certain threshold. After firing, the neuron resets and continues the process. The simulation also showed that increasing the input current leads to a higher firing rate.

This demonstrated how neuron behavior can be represented mathematically and visually.

## What I Learned
From this assignment, I learned how biological neuron behavior connects to artificial neural networks used in AI. I also gained experience using Wolfram Language to simulate and visualize data.

This helped me understand how neuroscience concepts are applied in artificial intelligence systems.

# Midterm – Bio-Inspired AI Audit Pilot

## Problem
The purpose of this midterm project was to evaluate ChatGPT using neuroscience concepts and compare its behavior to how the human brain processes information.

## Approach
To complete this project, I conducted multiple experiments focusing on key areas of cognition:

- Parallel Processing (Module 02)
- Memory and Learning (Module 04)
- Attention and Consciousness (Module 07)

Each experiment tested how ChatGPT responds to tasks that simulate human cognitive functions.

## Results
The results showed that ChatGPT can successfully complete tasks related to memory, attention, and reasoning. However, the system processes information sequentially rather than simultaneously, which differs from the human brain.

ChatGPT was able to recall information within a conversation, but it does not store memory the same way humans do. It also demonstrated the ability to prioritize instructions when multiple directions were given.

## What I Learned
From this midterm project, I learned that AI systems can simulate human cognitive behavior but do not function like biological systems. The human brain relies on complex neural processes, while AI systems depend on pattern recognition and data.

This project helped me better understand both the capabilities and limitations of artificial intelligence.

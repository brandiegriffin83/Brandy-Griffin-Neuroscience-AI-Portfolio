# Midterm – Bio-Inspired AI Audit Pilot

## Problem
The purpose of this midterm project was to evaluate ChatGPT using neuroscience concepts and compare its behavior to how the human brain processes information.

## Approach
To complete this project, I conducted several experiments based on course modules:

- Parallel Processing (Module 02)
- Memory and Learning (Module 04)
- Attention and Consciousness (Module 07)

Each experiment tested how ChatGPT responds to tasks that simulate human cognitive functions.

## Results
The results showed that ChatGPT was able to successfully complete all assigned tasks. However, the system processed information sequentially rather than simultaneously, which is different from how the human brain operates.

ChatGPT demonstrated strong memory recall within a conversation but does not store information like the human brain. It also showed the ability to prioritize instructions when multiple directions were given.

## What I Learned
From this project, I learned that artificial intelligence can simulate human cognitive behavior but does not function like biological systems. The brain relies on neural processes and experience, while AI depends on pattern recognition and data.

This project helped me better understand both the strengths and limitations of AI systems.

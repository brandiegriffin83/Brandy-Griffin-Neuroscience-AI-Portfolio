# L05 – Neuron Modeling Lab

## Problem
This lab focused on understanding how neuron behavior can be modeled using computational methods.

## Approach
I worked with neuron parameters and simulations to observe how neurons respond to different inputs. The lab involved analyzing how signals affect neuron activity.

## Results
The results showed how neurons respond to input signals and how their behavior can be represented through models.

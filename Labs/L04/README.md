# L04 – IoT Protocol Simulation

## Problem
This lab focused on understanding how IoT devices communicate using protocols like MQTT and CoAP.

## Approach
I set up the environment, installed required packages, and ran simulations to mimic IoT sensor data. I followed each step and documented the process using screenshots.

## Results
The simulation successfully showed communication between devices and allowed real-time data visualization.

## What I Learned
I learned how IoT systems communicate, how data flows between components, and how to troubleshoot setup issues.

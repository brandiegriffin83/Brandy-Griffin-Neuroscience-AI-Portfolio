# L07 – Memory and Attention in AI

## Problem
This lab focused on testing how AI systems handle memory and attention using neuroscience concepts.

## Approach
I conducted experiments to test:
- Working memory
- Memory recall over time
- Selective attention

## Results
The AI system was able to handle large amounts of information and follow instructions correctly. It showed strong recall within a conversation.

## What I Learned
I learned that AI memory is based on context rather than biological processes. AI can simulate attention, but it does not function the same way as the human brain.
